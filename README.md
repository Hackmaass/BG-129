# 🛠️ ForgeCart | Premium Developer Gear

![ForgeCart Banner](https://img.shields.io/badge/BackForge-2026-f97316?style=for-the-badge&logo=github)
![Tech Stack](https://img.shields.io/badge/Stack-HTML5%20%7C%20CSS3%20%7C%20JS-blue?style=for-the-badge)

ForgeCart is the official premium merchandise store for the **BackForge Hackathon 2026**. Designed with a focus on high-end developer aesthetics, it provides a seamless shopping experience for elite gear, from ergonomic apparel to exclusive digital assets.

---

## ✨ Features

- **💎 Premium Glassmorphism UI**: A state-of-the-art interface built with vanilla CSS, featuring deep blurs, mesh gradients, and subtle micro-animations.
- **🛡️ Developer-Centric Catalog**: Curated categories including *Premium Apparel*, *Digital Assets*, and *Hardware Accessories*.
- **🔍 Advanced Filtering**: Real-time search and category-based filtering to find your gear fast.
- **🛒 Dynamic Shopping Cart**: A fully integrated cart system to manage your "deployment" of gear.
- **🔐 Secure Authentication**: Clean, focused Login and Registration flows.
- **📦 Order Tracking**: History view for tracking your past acquisitions.

---

## 🚀 Tech Stack

- **Core**: Semantic HTML5 & Modern ES6+ JavaScript.
- **Styling**: Vanilla CSS3 with:
  - Custom Design System (CSS Variables)
  - CSS Grid & Flexbox Layouts
  - Advanced Backdrop Filters (Glassmorphism)
  - Keyframe Animations (`fadeInUp`, `pulseGlow`, `gradientShift`)
- **Typography**: [Outfit](https://fonts.google.com/specimen/Outfit) (Headings) & [Inter](https://fonts.google.com/specimen/Inter) (Body).
- **Icons**: [Font Awesome 6.4.0](https://fontawesome.com/).

## 🛠️ Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Edge, or Safari).

### Local Setup
1. **Clone the repository**:
   ```bash
   git clone https://github.com/backforgenmiet/Management-E-Commerce-Web-2.git
   ```
2. **Navigate to the directory**:
   ```bash
   cd Management-E-Commerce-Web-2
   ```
3. **Run the application**:
   Simply open `index.html` in your browser or use a live server extension.
